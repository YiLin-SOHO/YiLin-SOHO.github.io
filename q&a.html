常見問題

1. 為什麼釹磁鐵N45, N52等級價錢比N35貴?

一般大家較常聽到的強力磁鐵都是N35，所以會誤以為強力磁鐵等級都一樣，而事實上強力磁鐵分為許多等級，N45, N52的材料價格又比N35高許多，高出RMB幾十塊錢一公斤(比如N45一公斤在160元左右, N35在130元左右)
其次, 在性能上, N35磁力也是遠遠不及N45, N52。他們主要差在剩磁、矯頑力、最大磁能積不同。N35的磁力屬於釹鐵硼牌號中偏小的，數字越大磁力越強，價格相對也越高。

2. 磁鐵要多少高斯才算合格?

關於這問題一直以來都有很多人討論，他們會問到磁鐵多少高斯可以吸多重的東西，磁鐵多少高斯可以吸多遠，磁鐵要多少高斯才算合格? 但這些問題其實很不精確也不易解答
因為磁鐵可以一次吸多重物品，可吸多遠也難以計算，並不是磁鐵高斯越大，吸力就越強，包括磁鐵的規格、形狀、充磁方式、以及吸物件的方向、接觸面是哪面…都有密切影響。因此無法單靠表面高斯說明任何事
以下舉個例子，假設有個小磁鐵5000表高斯，能吸1.5KG。大磁鐵1000表高斯，反而能吸100KG。也就是說跟磁鐵的體積、接觸面都有關係，無法單從高斯去判斷磁鐵吸力。

3. 釹鐵硼方塊磁鐵比圓片磁鐵貴?為什麼?

一般最常見的磁鐵形狀，無非是圓片磁鐵，不然就是方塊磁鐵。
在報價上圓片磁鐵一定會比較貴，為什麼呢? 因為方塊磁鐵生產時要切3刀才能成形，花費的工時較多，成本也就越高，另外因為工時較長所以交期通常也會比圓片要久一些。
再深入講一些，因為工廠在製作方塊磁鐵時是使用方塊毛坯，在製作圓片磁鐵時是使用圓柱毛坯，這2種毛坯差別就是，如果是圓柱毛坯只要切一刀就成形了，而且還不會有廢料，但如果是方塊毛坯則要切3刀，且廢料會比較多些。

4. 釹鐵硼為什麼要加入金屬鏑?

釹鐵硼的耐溫性能本身較差，磁鐵工廠為了提高矯頑力，所以在配方中加入金屬鏑，從而提升釹鐵硼工作溫度
那如果釹鐵硼磁鐵少了鏑會怎麼樣呢?
[釹鐵硼圖片]
簡單來說，釹鐵硼如果缺少釹，磁力就會變弱(剩磁降低)，如果有鏑的高牌號永磁體缺少鏑，可能導致牌號降低、矯頑力降低，可以理解成耐高溫特性降低

釹鐵硼各牌號中含金屬鏑各占多少%?
M牌號: 0-1%
H牌號: 0-2%
SH牌號: 1-4.5%
UH牌號: 3-6%
EH牌號: 4-10%

5. 鐵氧體磁鐵如何切割?

不同於強力磁鐵，鐵氧體有著不導電的特性，因此無法進行線切割加工。鐵氧體只能對其外徑和厚度進行加工，不能分割。
如分割其原有性能會發生變化，會出現弱磁現象。外徑和厚度進行加工多為水磨加工，特殊狀況則會使用砂輪機研磨加工..等

6. 鐵氧體需要模具嗎?

由於鐵氧體生產要比釹鐵硼磁鐵複雜許多，交期也較長，因此我們平常看到的鐵氧體，即便只是個普通的圓片，
也都需要模具生產的，只因為工廠生產的量大且常規，所以才沒和客戶收模具費用，不然如果是一些不常見的尺寸，客戶就得花錢開模具了

7. 磁鐵高斯越高，吸力就越強嗎?

很多人常常會以為高斯值與吸力成正比，但事實上沒有直接關係。
如下以N35釹鐵硼磁鐵為例，厚度皆為1mm，吸附的物件皆為10mm厚的鐵片:

直徑5mm / 厚度1mm / 中心表磁1702高斯 / 最高表磁2139高斯 / 吸力0.281kg
直徑6mm / 厚度1mm / 中心表磁1485高斯 / 最高表磁1968高斯 / 吸力0.360kg
直徑8mm / 厚度1mm / 中心表磁1170高斯 / 最高表磁1711高斯 / 吸力0.537kg
直徑10mm / 厚度1mm / 中心表磁973高斯 / 最高表磁1557高斯 / 吸力0.735kg
直徑12mm / 厚度1mm / 中心表磁833高斯 / 最高表磁1332高斯 / 吸力0.907kg
直徑15mm / 厚度1mm / 中心表磁667高斯 / 最高表磁1068高斯 / 吸力1.136kg

以上量測數據，可以發現磁鐵高斯越高不代表吸力就越強喔!

8. 圓片和環形磁鐵差異?

尺寸:圓片標準尺寸為直徑和厚度，環形標準尺寸為外徑、內徑以厚度
價格:環形磁鐵比圓片磁鐵多了鑽孔的加工步驟，因此價格上會比較高一點
磁力:單以高斯值來說，環形的最大高斯值會比圓片大，但以實際吸力來說的話，圓片吸力會較高，因為圓片的受磁面積高於環形磁鐵

9. 磁鐵一定要開模具嗎? 開模具的費用及時間？

以目前主流燒結釹鐵硼來說，通常是不需要開模具的，而是使用成形塊料做加工，就連一些特殊形狀也可以透過加工完成而以鐵氧體來說，一般都是要開模具的，除非工廠剛好有公規模具適合客戶，才會跟客戶說不收模具費
鐵氧體模具的費用與模具大小及困難度有關，所以費用從1萬~1X萬不等
模具製作時間15~60天不等，具體時間得先和客戶溝通，了解需求後評估

10. 無刷電機裡用的磁鐵耐溫多少度?

先簡單介紹無刷馬達，無刷馬達採取電子換向，線圈固定，磁極旋轉，透過霍爾元件感應磁場變化，以送出正確的馬達控制訊號，讓馬達能夠持續穩定的運轉
那無刷電機中的磁鐵耐溫是多少度呢? 其實並不一定，不過大部分落在120-180度較多，
釹鐵硼牌號:
H(120°C Max)
SH(150°C Max)
UH(180°C Max)
EH(200°C Max)
AH(220°C Max)

11. 何謂高性能釹鐵硼?主要應用?

高性能釹鐵硼指的是內稟矯頑力(Hcj, Koe)與最大磁能積((BH)max, MGOe)的合大於60的燒結釹鐵硼
主要應用:包含傳統汽車產業、新能源汽車(驅動電機)、風力發電、變頻空調、節能電梯、工業機器人、消費性產品、磁浮列車、太陽能…等各種領域

12. 無刷電機為什麼很少用鐵氧體磁鐵呢?

因為無刷電機的電機功率跟磁鐵性能有著密切關係，簡單來說，釹鐵硼的體積與材料就已決定了電機的最大功率
因此大家熟知的鐵氧體，由於磁力性能較低的特性，所以很少被拿來用在無刷電機應用。
補充: 無刷電機最常見的磁鐵是燒結釹鐵硼(瓦片)及粘結釹鐵硼(環形)

13. 馬達磁鐵的磁性對電機應用有什麼影響?

磁力越強，扭力越大，功率高，但轉速低，常用於高電壓電機
磁力越弱，扭力越小，功率低，但轉速高，常用於低電壓電機
一般狀態下磁性越強越好(但超過線圈磁場強度一定值時，磁鐵磁性對電機馬達影響會下降)

14. 伺服馬達一定會用到磁鐵嗎? 可以用電磁鐵替代永磁嗎?

伺服電機分為交流永磁同步伺服及交流異步伺服，如果是交流永磁同步伺服就可能用到磁鐵，而且使用得種類也很多，最常見的就是釹鐵硼磁鐵
另外，是否可以用電磁鐵替代永久磁鐵呢?
其實要看伺服電機的類型，考量到電磁鐵需要電費，以及在有限的電流和空間內，能否產生的比永磁鐵還強的磁場，這些都是需要考量的

15. 同個磁鐵表磁高斯為什麼不一樣?

因為表磁高斯指的是磁鐵某個點量測到的數據，不可能每次測量到的位置都完全一樣，所以每次顯示的高斯值也會不同
。一般廠商會取最大表磁高斯的值來表示，而不是一般大家認為的中心高斯，所以在購買磁鐵時，需要和廠商確認清楚

16. 鐵氧體磁鐵噴漆跟烤漆哪個好呢?

兩種方法皆可以使用，但哪個比較呢? 我們認為烤漆比較好
因為烤漆的稜角平順，表面光滑平整，噴漆的稜角則比較粗糙，表面沒那麼光滑，摸起來稍微粗糙，另外耐用度烤漆也比噴漆表現好許多

17. 什麼是飽和充磁?

很多人以為磁鐵本身就有磁性，其是不是這樣的，磁鐵在加工過程中並沒有磁性，是在切片、倒角、電鍍後才進行充磁的，並且充磁需要一定的磁場強度。
一般來說充磁的磁場強度需為磁鐵本身矯頑力的5-7倍，若強度太低達不到飽和充磁，此時磁力會遠低於飽和充磁的磁鐵

另外如果飽和充磁後，產品會擁有該磁鐵材質的最大磁能積，但此最大磁能積已固定，如果不換磁鐵材質，就算提供更大的磁場強度也無法讓磁力提升

18. 釤鈷磁鐵可以精加工嗎?

一般都是聽到釹磁鐵可以精加工處理。不過釤鈷機械加工性能其實也很好，不過由於成本較高，使用機會也比釹磁鐵低

釤鈷磁鐵特色:釤鈷耐溫相當高，達300°C以上，且因為含鐵量低，不易生鏽，即便生鏽也只有一些鏽點，不會影響性能和使用，所以在一些特殊場域，釤鈷磁鐵是無可取代的。
